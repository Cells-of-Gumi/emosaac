## 프로젝트 소개

---

### 개요

- 삼성청년소프트웨어아카데미(SSAFY) 2학기 특화프로젝트(빅데이터 추천)에서 만든 프로젝트
- 2023.02.27  ~ 2023.04.07 (6주)
- 총 6명 ( 프론트엔드 3명, 백엔드 3명)으로 구성

### 주제

- 네이버, 카카오, 리디의 웹툰 / 웹소설을 한 곳에서 빅데이터 추천방식을 통한 사용자에게 추천해주는 플랫폼

### 아키텍처

- 찬희야 아키텍처 넣어주라
